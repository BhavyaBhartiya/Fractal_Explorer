# Fractal_Explorer

This is a Julia fractal explorer created entirely with HTML, CSS, and JavaScript. It does not rely on external libraries like math.js for complex number calculations, which helps to significantly reduce resource usage. As a result, it does not require WebGL or other GPU-based optimisations. This allows users to explore Julia fractals without needing a dedicated GPU or experiencing excessive lag. Furthermore, it minimises resource consumption by reducing the number of divisions and multiplications within loops and by utilising 32-bits for pixel colouring instead of 8-bits through Uint32Array.
