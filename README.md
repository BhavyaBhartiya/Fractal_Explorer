# Fractal_Explorer

This is a Julia fractal explorer created entirely with HTML, CSS, and JavaScript. It does not rely on external libraries, such as math.js, for complex number calculations, which significantly reduces resource usage. Additionally, it does not require WebGL or other GPU-based optimisations. Instead, it minimises resource consumption in various ways, including reducing the number of divisions and multiplications within loops and using 32-bit pixel colour values via Uint32Array instead of 8-bit. This allows users to explore Julia fractals without needing a dedicated GPU or experiencing excessive lag.

## Theme - Endless

I believe this project should be in the Endless theme as it explores fractals, which are a geometric pattern that remain similar no matter how much you zoom in, and allows the user to experience that interesting phenomenon themselves.
